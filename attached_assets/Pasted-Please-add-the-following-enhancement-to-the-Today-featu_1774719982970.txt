ADD A TODAY / TOMORROW TOGGLE

Directly below the header and date, add a simple pill-style toggle with two options: "Today" and "Tomorrow." This is the same screen, same feature — just two time modes.

Toggle design:
- Soft pill container with a warm off-white/light sand background (#f0ece6)
- Active tab: white background, dark text, very subtle shadow
- Inactive tab: no background, muted gray text
- Two words only. No icons. No heavy UI.

How it works:
- Default always opens on Today
- Switching to Tomorrow shows a completely separate list — same format, same micro Will list input, same circle + strikethrough interaction, same auto-save
- The two lists are stored independently: one keyed to today's date, one keyed to tomorrow's date
- Both views are fully editable at all times

Midnight transition logic:
- Tomorrow's list moves into the Today slot
- Old Today's list is deleted — consistent with the impermanence philosophy
- Tomorrow resets to blank
- The cycle: user plans Tomorrow the night before → wakes up and it's already seeded as Today → lives it → Tomorrow is blank again
