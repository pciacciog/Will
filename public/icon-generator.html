WILL App Icon Generator

Generate all required iOS app icon sizes for App Store submission

Instructions:

  1. Click "Generate All Icons" to create all required sizes
  2. Click "Download All Icons as ZIP" to get the complete icon set
  3. Extract the ZIP file and copy all PNG files to: ios/App/App/Assets.xcassets/AppIcon.appiconset/
  4. Delete the old AppIcon-512@2x.png file
  5. Build and upload your app to resolve the App Store rejection
